# BoatDetection
Репозиторий с готовым проектом системы искусственного интеллекта мониторинга судов в порту для практики (системы искусственного интеллекта) 

# Содержание проекта
* db - папка с файлом БД "history.db"
* detections - папка для сохранения результатов обработки
* static - папка для ресурсов сервера
  * css - папка с файлами стилизации CSS
  * icons - папка с иконками для веб-приложения
* templates - папка с файлами HTMl страниц веб-приложения
* test_data - папка с тестовыми данными

# Используемые средства
* Python 3.12.1
* Flask  3.1.3
* Ultralytics 8.4.75
* Openpyxl 3.1.5
